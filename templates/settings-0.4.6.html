How to use: (2 steps)

  1. Export and place the abExport.php file in your plugin directory.
  2. Add the following code to the top of your plugin, just after the plugin information comment. After the */ 
                    if (!function_exists('get_plugins')) {

                   require_once ABSPATH.'wp-admin/includes/plugin.php';
               }
               $plugin_folder = get_plugins('/'.plugin_basename(dirname(__FILE__)));
               $plugin_file = basename((__FILE__));

               require_once 'abExport.php';

That's it, Everything else is done by the plugin.
